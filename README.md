# Expense Tracker (Трекер расходов) 

## Автор
Захаров Никита Михайлович

## Описание программы
Приложение для учёта личных расходов с возможностью добавления записей, фильтрации по категориям и датам, подсчёта суммы за период. Данные сохраняются в формате JSON.

## Как использовать
Запустите expense_tracker.py.
Введите сумму (только положительное число), категорию и дату.
Нажмите «Добавить расход» — запись появится в таблице.
Используйте фильтры по категории и дате.
Для подсчёта суммы за период выберите даты и нажмите «Сумма за период».
Данные сохраняются автоматически.

## Примеры использования
Добавьте расход: сумма — 100.00, категория — «еда», дата — 2026-05-04.
Отфильтруйте по категории «еда».
Посчитайте сумму расходов за неделю.
